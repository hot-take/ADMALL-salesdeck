# American Dream — Interactive Sales Deck

A high-end, cinematic interactive presentation designed for the **American Dream** vision. This deck provides a premium, immersive experience showcasing the scale, luxury, and opportunity of North America's most ambitious destination.

## ✨ Features

- **Ethereal Entry Experience**: A sophisticated, light-themed landing page that sets the stage for the presentation.
- **Cinematic Transitions**: Custom GSAP-powered transitions that create a seamless, high-end "curtain" feel between slides.
- **Dynamic Themes**: A "Light-to-Dark" visual journey, transitioning from an airy intro to a deep, midnight-cinematic deck.
- **Premium Design System**: Glassmorphism cards, refined typography (Serif/Sans-serif mix), and a curated gold-and-charcoal color palette.
- **Interactive Components**: 
  - Dynamic statistics and fact cards.
  - Immersive video backgrounds.
  - Interactive brand and venue highlights with modal deep-dives.
- **Unified CTAs**: All primary call-to-action buttons and contact cards are unified to redirect to the official American Dream website.
- **Responsive & Optimized**: Fully responsive across desktop, tablet, and mobile, with an intelligent custom cursor system.

## 🛠️ Technology Stack

- **Core**: React 19 + Vite
- **Animations**: GSAP (GreenSock Animation Platform)
- **Icons**: Lucide-React
- **Styling**: Vanilla CSS (Custom Variable-based Design System)
- **Deployment**: Optimized for high-performance static hosting.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/hot-take/ADMALL-salesdeck.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/components`: Reusable UI components (Slides, Chrome, EntryScreen, etc.)
- `src/data`: Centralized slide content and modal metadata.
- `src/styles`: Global design system and component-specific styles.
- `public/images`: Optimized visual assets and brand identifiers.
- `public/videos`: Cinematic background media.

## 🎨 Design Philosophy

The deck is built on the principle of **"Quiet Luxury"**. It avoids cluttered interfaces in favor of large typography, significant negative space, and subtle micro-animations. The focus is always on the brand identity and the scale of the vision.

## 📝 Design & AI Note

### Design Rationale
The deck follows a **"Quiet Luxury"** philosophy, focusing on high-end typography and a cinematic **"Light-to-Dark"** transition. We move from an ethereal, airy entry screen into a deep, midnight-themed deck to allow the brand visuals and gold accents to take center stage.

### AI Integration
- **Asset Generation**: Used DALL-E 3 to generate custom architectural and interior visuals, ensuring a consistent cinematic lighting style across all slides.
- **Motion Prototyping**: AI assisted in crafting complex GSAP transition timelines, focusing on organic "curtain-reveal" effects.
- **Cleanup & Audit**: Leveraged AI for iterative code audits to ensure a minimal, efficient codebase and responsive layout.

### Future Improvements
With more time, I would focus on re-integrating high-performance 3D background interactions and implementing more advanced scroll-momentum physics for a truly tactile experience.
